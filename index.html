Condicionais

If representa um desvio no programa quando uma condição é satisfeita. Apresenta o seguinte formato:

instrução

if(condição) {
  instrução
}

instrução

No exemplo abaixo, a função executarCódigo() sempre é executada quando a variável x é igual à 5. Modifique o valor para verificar quando a função será ou não chamada.

int x = ;

if(x == 5) {
    executarCodigo();
}

Uma instrução else sempre é acompanhada de uma instrução if. Quando a condição do bloco if não é satisfeita, são executadas as instruções do bloco else. Apresenta o seguinte formato:

instrução

if(condição) {
    instrução
} else {
    instrução
}

instrução

Teste no exemplo interativo abaixo, teste a execução da função executarOutroCódigo(), lembrando que a condição do bloco if não pode ser satisfeita.

int x = ;

if(x > 5) {
    executarCodigo();
} else {
    executarOutroCodigo();
}

Pode ocorrer um aninhamento de condições. Nesse caso uma instrução if segue logo após à uma estrutura else. Apresenta o seguinte formato:

instrução

if(condição) {
	instrução
} else if {
      instrução
} else {
      instrução
}

instrução

No exemplo abaixo, a função executar() é executada, pois a condição do primeiro bloco if não é satisfeita.

int x = 2;

if(x > 5) {
	executarCodigo();
}
else if(x ==2) {
      executar();
}
else {
      executarOutroCódigo();
}

Existe também outra forma de aninhar ifs:

int x = 2;

if(x > 5) {
   executarCodigo();
} else {
   if(x == 2) {
       executar();
   } else {
       executarOutroCodigo();
   }
}

No exemplo anterior, aninha-se um if dentro do else sem a necessidade de um bloco else-if.

Um bloco switch é utilizado para concentrar várias instruções if. Somente tipos primitivos, Strings e enumeradores pode ser utilizados.

switch(variavel) {
    case condicao1:
        [break;]
    case condicao2:
        [break;]
    case condicao3:
        [break;]
    case condicao4:
        [break;]
    [default:]
}

A instrução break para a execução do bloco switch e sai para o fluxo normal do programa. A instrução default, indica que nenhuma das condições foram satisfeitas. Se não houver a instrução break, todas os blocos de códigos das condições subsequentes serão executadas, inclusive o bloco da condição default se houver. O exemplo abaixo mostra um swicth para dia da semana.

String diaDaSemana = "Domingo";

switch (diaDaSemana) {
    case "Domingo":
        System.out.println("Você informou o dia 1");
        break;
    case "Segunda":
        System.out.println("Você informou o dia 2");
        break;
    case "Terça":
        System.out.println("Você informou o dia 3");
        break;
    case "Quarta":
        System.out.println("Você informou o dia 4");
        break;
    case "Quinta":
        System.out.println("Você informou o dia 5");
        break;
    case "Sexta":
        System.out.println("Você informou o dia 6");
        break;
    case "Sábado":
        System.out.println("Você informou o dia 7");
        break;
    default:
        System.out.println("O dia informado não existe!");
        break;
    }
}

O operador ternário é uma maneira mais rápida de executar uma instrução if. É utilizado mais comumente em retornos de funções.

(codição) ? retorno-verdade : retorno-falso;

O exemplo a seguir retorna um String conforme a condição ilustrada por x==0.

return (x==0) ? “Encontrou” : “Não encontrou”;

Ao omitir o retorno, o operador ternário sempre retornará um booleano.

return (x==0);

Laços de Repetição

O comando for é um laço de repetição simples que exige 3 parâmetros:

  • Inicialização de uma variável
  • Condicional
  • Incrementador

Estrutura:

for(inicializacao ; condição ; incrementador ) {
    instrução;
}

O exemplo abaixo executa 1000 vezes um mesmo comando.

for(int i = 0; i < 1000 ; i++) {
    System.out.println(i);
}

O exemplo abaixo executa 500 vezes um mesmo comando, pois o incremental pula de dois em dois.

for(int i = 0; i < 1000 ; i+=2) {
    System.out.println(i);
}

O exemplo abaixo executa 1000 de trás para frente.

for(int i = 1000; i >=0 ; i--) {
    System.out.println(i);
}

O comando while executará as instruções em seu bloco até a condição ser satisfeita. A condição deve ser controlada dentro do corpo do bloco while. Se a condição não for satisfeita, a condição while nem é executada.

while(condicao) {
     instrução;
}

O exemplo a seguir é um acumulador que soma todos os números de 1 a 100.

int soma = 0;
int contador = 1;

while(contador != 100) {
    soma += contador;
    contador++;
}

System.out.println(soma);

Cuidado! O exemplo abaixo deve ser evitado, pois se trata de um loop infinito!


while(true) {
    System.out.println(“Nunca mais vou sair daqui!”);
    System.out.println(“Tem que dar um jeito de abortar o programa.”);
}

A instrução do-while é similar à instrução while. A diferença é que as instruções dentro deste bloco de repetição são executadas pelo menos uma vez mesmo que a condição não seja satisfeita.

do {
     instrução;
} while(condicao);

No exemplo a seguir as instruções do bloco do-while são executadas pelo menos uma vez e o fluxo segue seu caminho normal:

int i = 5;

do {
     System.out.println(i);
} while(i > 5);

No exemplo a seguir as instruções do bloco do-while são executadas até o final:

int i = 5;

do {
     System.out.println(i);
     i--;
} while(i > 0);
