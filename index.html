If representa um desvio no programa quando uma condição é satisfeita. Apresenta o seguinte formato:

instrução

if(condição)
{
  instrução
}

instrução

No exemplo abaixo, a função executarCódigo() sempre é executada.

  int x = ;

  if(x == 5) {
    executarCodigo();
  }

Uma instrução else sempre é acompanhada de uma instrução if. Quando a condição do bloco if não é satisfeita, são executadas as instruções do bloco else. Apresenta o seguinte formato:

instrução

if(condição)
{
    instrução
}
else
{
    instrução
}

instrução

No exemplo abaixo, a função executarOutroCódigo() é executada, pois a condição do bloco if não é satisfeita.

int x = ;

if(x > 5) {
    executarCodigo();
}
else
{
    executarOutroCodigo();
}
